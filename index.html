5. 🧠 AI & Machine Learning Processing

Our App uses artificial intelligence and machine learning for enhanced functionality:

5.1 On-Device AI Features

The following AI features process data entirely on your device — no data is sent to external servers:

  • 🔍 Auto Edge Detection: Automatically detects document boundaries in camera preview
  • ✂️ Auto Crop: Crops and straightens scanned documents
  • 🎨 AI Enhancement: Automatically adjusts brightness, contrast, and removes shadows
  • 📝 OCR Text Recognition: Extracts text from images (English, Hindi, and other languages)
  • 📷 QR/Barcode Scanning: Reads QR codes and barcodes
  • 🏷️ Document Classification: Identifies document type (ID card, invoice, receipt, etc.)
  • 🌐 On-Device Translation: Translates extracted text between languages
  • 🔍 Entity Extraction: Detects phone numbers, emails, and addresses in text
  • 📊 Text Summarization: Creates summaries of extracted text
  • 🌍 Language Detection: Identifies the language of extracted text
🛡️ Privacy Guarantee: All AI processing uses Google ML Kit's on-device APIs. Your documents, images, and extracted text are NEVER sent to Google, our servers, or any third party for AI processing.

5.2 AI Model Downloads

Some AI features (such as translation) require a one-time download of language models. These models are downloaded from Google's servers and stored on your device. Only the model files are downloaded — your documents and text are NOT uploaded during this process.

5.3 Google ML Kit

We use Google ML Kit for on-device machine learning. Google ML Kit's on-device APIs process data locally and do not send your data to Google servers. For more information, please refer to Google ML Kit Terms of Service .

7. 🔐 Data Storage & Security

7.1 Local Storage

All user-generated content is stored locally on your device:

  • 📄 Scanned documents and PDFs → Stored in app's private storage
  • 🖼️ Scanned images → Stored in app's private storage
  • 📋 Scan history → Stored in local SQLite database
  • ⚙️ App settings → Stored in SharedPreferences
  • 📷 QR scan history → Stored in local SQLite database

7.2 Security Measures

We implement the following security measures:

  • ✅ All data is stored in the app's private directory, inaccessible to other apps
  • ✅ No cloud storage or server-side storage of user documents
  • ✅ ProGuard/R8 code obfuscation in release builds
  • ✅ HTTPS encryption for all network communications (ads, model downloads)
  • ✅ No logging of sensitive user data

7.3 Data Deletion

You have full control over your data. You can delete any scanned document, PDF, or history entry directly within the App. Uninstalling the App will permanently delete all locally stored data.
